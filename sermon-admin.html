Admin Guide

How to Add Sermons

  1. Click "Add New Sermon" tab
  2. Fill in all required fields (marked with *)
  3. Upload files to your server in the /sermons/ folder
  4. Enter the file paths in the form
  5. Click "Save Sermon"
  6. Sermon will appear on the Sermons page immediately

File Upload Instructions

  • Audio Files: MP3, WAV, OGG (recommended: MP3)
  • Video Files: MP4, WebM (recommended: MP4)
  • Upload Location: Create a /sermons/ folder in your website root
  • File Names: Use simple names: message-audio.mp3
  • File Paths: Enter as: sermons/message-audio.mp3

YouTube Links

  • Go to YouTube video
  • Click "Share" → "Embed"
  • Copy the src URL: https://www.youtube.com/embed/VIDEO_ID
  • Paste in the YouTube Link field

Data Management

  • Export: Backup your sermons as JSON file
  • Import: Restore sermons from JSON backup
  • Auto-save: Changes save automatically to browser
  • Persistence: Data survives browser restarts
  • Clear Data: Only clears if you delete browser data

Viewing Sermons

  • Go to: Sermons page from navigation
  • Users can search sermons by title or speaker
  • Users can filter by category and date
  • Users can click sermons to read full content
  • Users can download audio/video files
  • Users can watch videos or listen to audio directly
